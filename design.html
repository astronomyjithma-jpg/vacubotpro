HOW IT WORKS

Three ways to
clean, one robot.

VacuBot Pro X1 switches between object avoidance, line follow, and manual WiFi control — while it vacuums dust and logs room temperature and humidity as it goes.

Control Modes

Three ways to
drive the robot

Switch modes depending on the room, the floor plan, or whether you just want to take the wheel yourself.

MODE / 01

Object Avoidance

Infrared and bump sensors detect furniture, walls, and obstacles in real time, steering the robot around them without getting stuck.

MODE / 02

Line Follow

An IR line sensor tracks a marked path on the floor, letting the robot follow a fixed route through a room or between rooms.

MODE / 03

WiFi Control (Manual)

Drive the robot yourself from a phone or browser over WiFi — forward, back, left, right — for spot cleaning exactly where you want it.

obstacle_avoid.ino

MODE / 01
// Object avoidance — IR + bump sensor loop
void loop() {
  int distance = readIRSensor();
  if (distance < THRESHOLD_CM) {
    stopMotors();
    reverse(200);
    turn("right", 300);
  } else {
    driveForward();
  }
  logTelemetry(readTemp(), readHumidity());
}

line_follow.ino

MODE / 02
// Line follow — dual IR line sensor
void followLine() {
  int left = analogRead(IR_LEFT);
  int right = analogRead(IR_RIGHT);
  if (left > right) turn("left", 50);
  else if (right > left) turn("right", 50);
  else driveForward();
}
